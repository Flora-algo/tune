# Tune

**调 Tune — 不练身材，调状态。**

一款极简的 iOS 状态调节 App，帮助你在情绪、睡眠、专注之间找回节奏。

---

## 关于 Tune

身体需要训练，状态需要调节。

Tune 不是健身 App，也不是冥想 App。它是一个安静的工具，陪你在一天的起伏里，把情绪、睡眠和专注，调到刚刚好。

- **情绪** — 在波动里找到平衡
- **睡眠** — 在喧嚣里找回安静
- **专注** — 在分心时回到当下

---

## 重要说明

Tune 是一款日常状态调节工具,不是医疗器械,也不提供医疗、心理或健康诊断服务。

- App 内的内容仅供参考,不能替代医生、心理咨询师或其他专业人士的建议
- 如果你正在经历持续的情绪困扰、睡眠障碍或健康问题,请及时寻求专业帮助
- 如遇紧急情况,请立即拨打当地急救电话或前往最近的医疗机构

照顾好自己,是这件事最重要的部分。

---

## 隐私

Tune **完全不收集任何用户数据**。

- 不收集健康数据
- 不收集位置信息
- 不收集账号信息
- 不发送任何数据到云端
- 没有第三方分析、没有广告 SDK

所有数据都只保存在你的设备本地。卸载 App，数据随之消失。
